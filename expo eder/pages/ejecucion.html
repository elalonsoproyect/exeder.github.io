duration of resistance tests



If this is a 24/7 project, ideally a 72 hour stress test should be performed.

Therefore, we must test the system exposed to significant load for a period slightly longer than a weekend.




Why perform stress tests?



The main goal of performing this test is to identify possible memory leaks. So during this test, memory utilization is closely monitored.
A memory leak is a failure of a software program to free discarded memory, resulting in poor performance or crash
